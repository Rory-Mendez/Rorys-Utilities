# Roadmap

This roadmap describes planned directions for Rory's Utilities. Nothing here is guaranteed or scheduled. Features are added when they are useful, stable, and fit the mod's philosophy: small, fast, compatible.

---

## Released

### v0.1.0
- Ctrl Sprint

---

## Planned

### Custom Keybinds
Allow each feature's key to be rebound in-game via Options > Controls, with the config file kept in sync.

### Quality-of-Life Utilities
Small client-side improvements:
- Toggle sprint (press once to sprint, press again to stop)
- Auto-jump toggle
- Coordinate display (HUD overlay, togglable)

### Configuration Improvements
- In-game config GUI (Forge mod options screen)
- Per-world config profiles

### Better Compatibility
- Test and document compatibility with common 1.2.5 mods (OptiFine, Too Many Items, etc.)
- Graceful degradation when conflicting mods are detected

### Rory's Ecosystem Integration
Future integration with other mods in the Rory's Mods ecosystem. Details to be announced.

---

## Out of Scope (Forever)

- Server-side gameplay changes
- Any feature that requires bytecode patching of `minecraft.jar`
- Anything that gives a competitive advantage in PvP or that could be considered a cheat

---

*This roadmap is updated as priorities change. Check the [CHANGELOG](../CHANGELOG.md) for what has actually shipped.*
